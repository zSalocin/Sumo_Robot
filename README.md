# Sumo_Robot
## Ideias

### Estrutura do robo
#### Dois motores de potencia
- Impulso via 2 motores
- 2 rodas e uma esfera
- Controle via ponte H

### Servo Motor
- Motor de potencia com eixo duplo
- Servo motor para controle
- 3 Rodas

## Estrutura do Programa
- Detecção de Borda
- Detecção de inimigo
- Decisão de estrategia
